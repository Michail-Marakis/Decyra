# During the Semester we did these presentations in front of the class.   

**Requirements Analysis & UX**  

This presentation focuses on the functional and non-functional requirements of the DECYRA application, which acts as a digital mentor for Computer Science students.    
It outlines the core user experience (UX) goals, details target user personas, and presents key field research findings gathered from a survey of 53 university students.

---

**Application Design**  
     
This presentation illustrates the architectural design and system interaction flows through detailed use cases, activity diagrams, and full system flowcharts.      
It also covers the application's physical design system including the color palette and typography alongside the evolution from low-to-high fidelity user interface prototypes.

---

**Implementation Analysis**        
     
This presentation breaks down the technical execution of the project, mapping out the full technology stack (Kotlin, Jetpack Compose, Java, Node.js) and the relational database schema.                
It provides a deep dive into the AI response pipeline (RAG), Face ID authentication via computer vision with the help of Augmented Reality, the integration of cloud architectures (AWS, Firebase, Supabase), and the engineering    challenges overcome during production scaling.
