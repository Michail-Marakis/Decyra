# Forum Module

This module implements the community review and discussion functionality of DECYRA, allowing users to browse academic reviews, filter content, like posts, and participate through comments.

It is built with **Kotlin** and **Jetpack Compose** for the UI and uses **Firebase Realtime Database** to manage reviews, likes, comments, views, and filtering data such as countries and universities. The module also supports **voice-to-text input** for review creation and standard Android notification services for user feedback.
