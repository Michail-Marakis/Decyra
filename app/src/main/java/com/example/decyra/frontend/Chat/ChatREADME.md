# Chat Module

This module implements the messaging functionality of DECYRA, allowing users to browse conversations and communicate through real-time one-to-one chat.

It is built with **Kotlin** and **Jetpack Compose** for the Android UI and uses **Firebase Realtime Database** to store conversations, messages, and user-related chat data. The module also supports **voice-to-text input** through Android Speech Recognizer and notification delivery through Firebase Messaging.
